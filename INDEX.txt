7️⃣ counters-handler.js (الملف الأساسي #1)
   ├─ معالج العدادات المستقل والكامل
   ├─ التحديث الفوري للعدادات
   ├─ دعم Offline-First
   ├─ المزامنة الذكية مع Firebase
   ├─ مكتوب بـ Vanilla JavaScript
   ├─ تعليقات عربية شاملة
   ├─ دوال رئيسية:
   │  ├─ updateCounter()
   │  ├─ loadCountersFromFirebase()
   │  ├─ displayCounters()
   │  ├─ updateSyncIndicator()
   │  └─ resetCountersForNewDay()
   ├─ جاهز للاستخدام الفوري
   ├─ الحجم: 12 كيلوبايت
   └─ المسار المقترح: /Al-Shafaq-Al-Ahmar/src/js/counters-handler.js

8️⃣ firebase-rules.json (الملف الأساسي #2)
   ├─ قواعد Firebase للأمان الجديدة
   ├─ فصل صارم بين أنواع البيانات
   ├─ فحص أمان كامل للبيانات
   ├─ بدون رسائل خطأ للعمليات المعلقة
   ├─ الأقسام الرئيسية:
   │  ├─ config (إعدادات المحطة)
   │  ├─ users (المستخدمون)
   │  ├─ shifts (الورديات)
   │  ├─ inventory (العدادات)
   │  ├─ supply (الإمدادات)
   │  ├─ activityLog (سجل النشاط)
   │  ├─ snapshots (النسخ اليومية)
   │  └─ syncQueue (قائمة الانتظار)
   ├─ جاهز للنشر في Firebase Console
   ├─ الحجم: 9.2 كيلوبايت
   └─ الهدف: Firebase Realtime Database → Rules
